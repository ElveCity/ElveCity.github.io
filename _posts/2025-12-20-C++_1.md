# 함수 오버로딩에 관하여

같은 이름의 함수 여러 개를 정의, 매개변수의 형태가 다를 때 서로 다른 함수로 구분지어 사용하는 것  
동일 행동을 하되, 상황에 따라 입력이 다른 경우라고 생각하면 된다.  
ex) 출력을 할건데 리턴타입을 int로 할건지, float, double로 할건지, char로 할건지.. 같은 때에 어차피 출력은 할거고 이름만 바꿔쓰자는 그런느낌

### C++ 기준 오버로딩 조건은 다음과 같다

1. 오버로딩이 되려면 파라메터 리스트가 달라야 한다.
2. 파라메터 타입이 달라야 한다.
3. 파라메터 타입의 순서가 달라야 한다.

### 안되는 경우

1. 리턴타입만 다를 때
2. 파라메터 이름만 다를 때

   #include <iostream>
   using namespace std;

   int add(int a, int b)
   {
   return a + b;
   }
      
   double add(double a, double b)
   {
   return a + b;
   }
      
   int add(int a, int b, int c)
   {
   return a + b + c;
   }
      
   int main()
   {
   cout << add(3, 4) << '\n';
   cout << add(2.5, 1.5) << '\n';
   cout << add(1, 2, 3) << '\n';
   }
