# Operating System_04
***
   
## System Call. 시스템 호출
   
사용자 프로세스가 커널에 서비스를 요청하는 인터페이스.
   
## Scheduling. 스케줄링
   
CPU 자원을 어느 프로세스에 할당할지 결정하는 정책
   
## Copy-on-write. COW, 쓰기 시 복사
   
프로세스를 복제할 때 메모리를 즉시 복사하지 않고 공유하다가, 어느 한 쪽에서 
쓰기 작업을 수행할 때만 실제 복사본을 만드는 메모리 절약 기법.
   
## Kernel Threads. 커널 스레드
   
프로세스와 동일한 주소 공간을 공유하며 독립적으로 실행되는 단위.
   
## Extent-based File System. 익스텐트 기반 파일 시스템
   
파일의 데이터 블록을 개별 포인터로 관리하는 대신, 시작블록과 길이 쌍인 익스텐트 단위로 
관리해서 메타데이터 효율을 높이는 방식.
   
## Journaling. 저널링
   
파일 시스템의 변경 사항을 실제 데이터 영역에 쓰기 전 전용 로그(Journal)에 
먼저 기록하는 기법.
   
## File System Checker(fsck)
   
파일 시스템의 무결성을 검사하고 일관성이 깨진 부분을 찾아 복구하는 도구.
